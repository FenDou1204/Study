# Markdown 文本中使用表情

编写 Markdown 文本时，适当插入表情能够增加文章趣味性，能够增强文章章节表达性，同时也能突出某些重点。如何插入表情？表情其实是一些指定的unicode 编码，通过指定语法即可引用相应表情。

## 使用表情语法

语法: `&#xCODE;`

其中，CODE 可以从下面两个链接中查到。

例如:

&#x1F600;-> `&#x1F600;`

&#x1F4D6;-> `&#x1F4D6;`
